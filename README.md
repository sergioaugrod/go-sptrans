# go-sptrans #

go-sptrans is a Go client library for the [SPTrans Olho Vivo API](http://www.sptrans.com.br/desenvolvedores/APIOlhoVivo.aspx).

### Features

- [ ] Routes 
- [ ] Stops
- [ ] Corridors 
- [ ] Companies
- [ ] Position of vehicles
- [ ] Arrival Forecast 
- [ ] Speed on lanes


## Install

go get github.com/sergioaugrod/go-sptrans/sptrans
